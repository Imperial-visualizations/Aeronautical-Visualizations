Visualisation Guidance

Hello :-)
This is the guidance for dynamic system visualisation.

×

Slider inputs control the parameters of the dynamic system shown.

×

Combination of mode shapes give system response. Drag the slider to leftmost for pure first mode response, vise versa.

×

Reset button set time to 0s and parameters to default values.

×

System responses are presented in plots. Hide and show responses by clicking legends.

×

Have fun!
