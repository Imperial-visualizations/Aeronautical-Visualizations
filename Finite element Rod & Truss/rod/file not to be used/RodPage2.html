The equation of motion of the rod modelling the steady state vibration can be written as following: $$ \mathbf{M}\ddot{q} + \mathbf{K}q = 0 $$ where $\mathbf{M}$ represents for mass matrix and $\mathbf{K}$ for global stiffness matrix. To simplify the following calculation, a unit mass and stiffness matrix is written out instead as following: $$ \frac{\rho A l_0}{6}\mathbf{M}^*\ddot{q} + \frac{EA}{l_0}\mathbf{K}^* q = 0 $$

By assuming the natural modes of vibration in the form: $$\mathbf{q} = \bar{q} e^{i\omega t}$$ we get the following differential equation: $$ \left ( -\omega ^2 \mathbf{M^*} + \frac{6E}{\rho l_{0}^{2}} \mathbf{K}\right )\bar{q} = 0 $$

Rearraging the equation to generalized eigenvalue problem, the equation subsequently transforms into: $$ \left ( \mathbf{K}^* - \lambda \mathbf{M}^* \right ) \bar{q} = 0  ,   \lambda = \frac{6E}{\rho l_{0}^{2} \omega^2 }$$

By solving the eigenvalue problem we get $\mathbf{k_{max}+1}$ egen values and the same number of eigen modes. By applying FEA, the vibration problem can be described by real symmtric matrices. It can be seen that the first eigenvalue is 0, which indicates the corresponding eigenmode as rigid body motions due to that rigid body DoF of the system. In this case we only have 1 zero eigenvalue is due t the fact that the rod is not constrained and can only move logitudinally.
