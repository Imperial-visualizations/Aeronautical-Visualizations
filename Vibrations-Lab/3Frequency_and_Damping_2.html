Visualisation Guidance

This is the guidance for Vibrations Laboratory, experimental data aquisition visualisation.

×

The four input boxes allow you to input your data one by one, for each different measurement. You need to input the "Load Amplitude Factor" only once, at the beginning, as it remains constant.

×

If no data is loaded, the "Submit" button will automatically load previous experimental data, that you can use to compare yours. When you fill all the three text boxes with data (Frequency, Amplitude and Phase Angle), the measurements will simply be added and displayed on the plots as you load them.

×

Use the reset buton to delete all the data that you have imported.
